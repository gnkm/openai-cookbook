# MongoDB Atlas Vector Search

[Atlas Vector Search](https://www.mongodb.com/products/platform/atlas-vector-search)は、MongoDB内で高次元ベクトルデータを効果的にインデックス化し、高速なベクトル類似性検索を実行するプロセスを簡素化する、フルマネージドサービスです。Atlas Vector Searchを使用することで、新しいプロジェクトでMongoDBをスタンドアロンのベクトルデータベースとして使用したり、既存のMongoDBコレクションにベクトル検索機能を追加したりできます。Atlas Vector Searchを使用すると、主要なパブリッククラウド（AWS、Azure、GCP）でベクトル検索の強力な機能を活用でき、FedRamp、SoC2コンプライアンスなどの規定によりエンタープライズ対応でありながら、大規模なスケーラビリティとデータセキュリティを標準で実現できます。
