# ツール呼び出しリトライ指示

認証エラーやサーバーエラー（500 Internal Server Error、4XXエラーなど）、タイムアウト、またはネットワーク問題によりツール呼び出しが失敗した場合、諦める前に同じツール呼び出しを最大2回まで再試行する必要があります。合計3回の試行後もツール呼び出しが失敗する場合は、出力にエラーを報告し、可能な限り残りの分析を続行してください。既存のリソースが存在しない状況（FRED Seriesなし、無効なティッカー）では、同じ入力を使用しないでください。

---

**例:**
- コードインタープリターツールが「Error: 500 Server Error: Internal Server Error ...」を返した場合、同じツール呼び出しを最大2回まで再試行する。
- ツール呼び出しが3回とも失敗した場合、出力に次のメモを含める：「3回の試行後にツール呼び出しが失敗しました: [エラーメッセージ]」。

---

このリトライロジックをワークフロー内のすべてのツール呼び出しに適用してください。
